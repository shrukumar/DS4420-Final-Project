## DS4420 Machine Learning and Data Mining 2 Final Project by Shrutika Kumar, Gabrielle Bambalan, and Jackson Goodman
#### Status: In Progress

This project aims to incorporate 3 different predictive models to analyze runner data.
1) Time-series/ARIMA to predict running logistics like distance and duration
2) User-User Collaborative Filtering to find most similar runners and form running-buddy recommendations
3) Bayesian Modeling to predict marathon finish times


### Data:
Data sourced from a [Kaggle Dataset on Long Distance Running](https://www.kaggle.com/datasets/mexwell/long-distance-running-dataset?select=run_ww_2019_m.csv). Files are within the data folder.


### How to Run:
Install dependencies and run notebooks. Each notebook operates separately and can be run in any order.
